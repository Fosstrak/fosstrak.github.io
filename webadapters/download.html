System Requirements

JDK 1.6.x
EPCIS Requires a standard compliant EPCIS such as the Fosstrak EPCIS to be installed.
Application Server Needs to be deployed in a standard Java Application Server. Tested with Tomcat 6.* and Glassfish V 3.*
Operating System No minimum requirements. Tested on Windows XP, and GNU linux.
